
The UI is a classic 3-column layout, similar to Slack or Discord, with a dark theme for navigation and a light theme for the main content.

### 1. Overall Layout (Root Component)

The layout is a top-level `flex flex-row` container dividing the screen into two main parts:
* **Column 1: `MainSidebar` (Pane 1):** A very narrow, dark purple sidebar on the far left.
* **Column 2: `MainArea`:** This takes up the remaining space (`flex-1`) and is a `flex flex-col` container. It is further divided into:
    * **Row 1: `TopHeader` (Pane 4):** A horizontal header bar.
    * **Row 2: `ContentArea`:** This takes the remaining vertical space (`flex-1`) and is itself a `flex flex-row` container, split into:
        * **`NavSidebar` (Pane 2):** The secondary, wider sidebar.
        * **`ChannelContent` (Pane 3):** The main content area.

---

### 2. Pane 1: `MainSidebar` (Far Left)

This component is a narrow (`w-20` approx.) `flex flex-col` with a very dark purple background (e.g., `bg-indigo-950`).

* **Top:** A rounded square (`rounded-lg`) container with the letters "SS" (likely a workspace logo).
* **Middle (Navigation):** A `flex flex-col` list of navigation items. Each item is text-only, white, and centered.
    * "Home"
    * "DMs"
    * "Activity"
    * "Files"
    * "More"
* **Bottom:** A circular avatar (`rounded-full`) with a red background (`bg-red-500`) and the letter "D" in white.

### 3. Pane 4: `TopHeader`

This component is a horizontal `flex items-center justify-between` bar with a dark purple background (`bg-purple-800`) and a `border-b border-purple-700`.

* **Left Side:** Two faded (e.g., `text-gray-400 opacity-50`) icons:
    * **Lucide Icon:** `ArrowLeft`
    * **Lucide Icon:** `ArrowRight`
* **Center (Search Bar):** A wide, rounded (`rounded-md`) search input with a slightly darker purple background (e.g., `bg-purple-700`).
    * It's a `flex items-center px-2`.
    * **Left Icon:** `Search` (Lucide)
    * **Input:** A text input with white placeholder text: "Search Scaler School Of Technology".
    * **Right Icon:** `SlidersHorizontal` (Lucide)
* **Right Side:** A `flex items-center gap-4` with light-colored icons (`text-gray-200`).
    * "94" (text)
    * A circular user avatar (image).
    * A "Huddle" component (text) with a `Headphones` icon (Lucide).
    * A `MoreVertical` icon (Lucide).
